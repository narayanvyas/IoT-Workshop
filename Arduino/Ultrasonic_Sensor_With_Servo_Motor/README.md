# Ultrasonic Sensor With Servo Motor
Ultrasonic Distance Sensor HC-SR04 With Servo Motor Tower Pro SG90.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. HC-SR04 Ultrasonic Distance Sensor
4. Servo Motor
5. Breadboard / PCB
6. LED
7. Buzzer
8. Jumper Wires
