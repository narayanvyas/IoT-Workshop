# Flame Detector
Flame Detector With Flame Sensor And Arduino.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. Flame Sensor
4. Buzzer
5. Breadboard / PCB
6. 2 x LED
7. 2 x 220 Ohm Resistors
8. Jumper Wires
